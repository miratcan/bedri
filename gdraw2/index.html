WordCanvas: Image-to-Text Renderer

Options

Adjust the minimum and maximum font sizes for the text. The maximum size is automatically set to 10% of the image width when an image is loaded.

Control the transparency of the text. Lower values make the text more transparent, higher values make it more opaque.

Enable to use only white text. When disabled, text will use colors from the original image.

Specify the number of iterations for word placement optimization. Higher values may improve the result but increase processing time.

Specify the total number of words to be placed on the image. More words will create a more detailed result but take longer to process.
